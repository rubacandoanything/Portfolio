Karighars

Architecture Dashboard

Designed a complete internal dashboard for Karighar, one of Bangalore’s leading interior design and turnkey project companies. This dashboard was created to streamline the company’s internal operations, improve project visibility, and help teams manage daily tasks more efficiently. this design focused on simplifying complex workflows, creating a clean visual hierarchy, and building a system that allows different departments—design, execution, vendors, accounts, and project managers—to collaborate smoothly. The result was a modern, intuitive, and highly functional dashboard that reduced friction in daily operations and improved overall productivity.
